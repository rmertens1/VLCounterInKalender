Anleitung zur Nutzung des Kalendertools


Vorgehen zum Zählen Ihrer Veranstaltungen

Bitte wählen Sie den Zeitraum aus, für den Termine aus Ihrem Kalender ausgelesen werden sollen. Sie können bestimmen, welche Termine nummeriert werden sollen, indem Sie sie markieren. Dabei können Sie ganze Veranstaltungen oder einzelne Termine markieren. Wenn Sie die Farben/Kategorien der Termine anpassen möchten, dann wählen Sie unter einer Veranstaltung einfach die gewünschten Kategorien aus. Und bestätigen die Auswahl Kategorien der Termine aktualisieren
Mit einem Klick auf 'Termine aktualisieren' werden die Neuerungen an Outlook übertragen. Dieser Prozess kann je nach Anzahl der ausgewählten Termine einige Minuten dauern.

Sollte es dazu kommen, dass Termine, die eigentlich zusammen gehören, getrennt voneinander angezeigt werden,
dann müssen Sie diese manuell in Ihrem Kalender umbenennen, sodass sie den identischen Namen besitzen.
