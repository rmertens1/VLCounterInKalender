Vorgehen zum Zählen Ihrer Veranstaltungen

Bitte wählen Sie den Zeitraum aus, für den Termine aus Ihrem Kalender ausgelesen werden sollen.
Sie können bestimmen, welche Termine nummeriert werden sollen, indem Sie sie markieren.
Mit einem Klick auf 'Termine aktualisieren' werden die Neuerungen direkt in Ihrem Kalender übertragen.

Sollte es dazu kommen, dass Termine, die eigentlich zusammen gehören, getrennt voneinander angezeigt werden,
dann müssen Sie diese manuell in Ihrem Kalender umbenennen, sodass sie den identischen Namen besitzen.
